Base URL

/api/v1

Response Format

{
  "success": true,
  "data": { /* endpoint specific data */ }
}

Endpoints

Home Page

GET /api/v1/home

Returns spotlight, trending, top airing, latest episodes, genres, and more.

Example:
/api/v1/home

Search Anime

GET /api/v1/search
Query Parameters:
keyword (required) - Search keyword
page (optional) - Page number (default: 1)
Example:
/api/v1/search?keyword=naruto&page=1

Search Suggestions

GET /api/v1/search/suggestion
Query Parameters:
keyword (required) - Search keyword
Example:
/api/v1/search/suggestion?keyword=one

Episode Servers

GET /api/v1/servers
Query Parameters:
id (required) - Episode ID

Returns available streaming servers for an episode (sub, dub, raw).

Stream Links

GET /api/v1/stream
Query Parameters:
id (required) - Episode ID
server (required) - Server name (e.g., "hd-1", "hd-2")
type (optional) - Type: "sub", "dub", or "raw" (default: "sub")

Returns HLS streaming links, subtitles, intro/outro timestamps.

Configuration

Set environment variables in .env.php:

ORIGIN=*
UPSTASH_REDIS_REST_URL=your_redis_url
UPSTASH_REDIS_REST_TOKEN=your_redis_token
RATE_LIMIT_WINDOW_MS=60000
RATE_LIMIT_LIMIT=100

JavaScript Client Library

Use our vanilla JavaScript client library (no dependencies!) to easily consume the API from the browser:

<!-- Include the client library -->
<script src="/public/js/hianime-api-client.js"></script>

<script>
// Initialize the client
const api = new HiAnimeAPI('/api/v1');

// Get home page data
api.getHome().then(data => {
    console.log(data.spotlight);
    console.log(data.trending);
});

// Search anime
api.search('naruto', 1).then(data => {
    console.log(data.results);
});

// Get stream links
api.getStream('anime-id?ep=1', 'hd-1', 'sub')
    .then(stream => {
        console.log(stream.link.file); // HLS URL
        console.log(stream.tracks);    // Subtitles
    });
</script>
